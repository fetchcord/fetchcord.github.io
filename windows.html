Installing on Windows

> Installation


To Install fetchcord, run python -m pip install fetchcord neofetch-win

Alternatively, you can use the neofetch package from scoop as well (show more info at the expense of possible GPU detection, for now).

Ensure that you have Python Installed & Added to PATH (you do this from the installer, or it can be done manually) - this should ensure that this command works just out of the box as is


> Run


Once installed, simply run fetchcord.
